# Contributing

This repository is a catalog of system prompts, tooling notes, and reference material for AI coding tools.

## Scope

Contributions should be limited to one of these categories:
- adding or updating a tool or provider folder
- correcting prompt text or metadata
- improving repository navigation and documentation
- fixing broken links, naming inconsistencies, or duplicate content

## Content Rules

- Keep source material attributable and clearly labeled.
- Do not commit secrets, tokens, or private prompt exports.
- Use clear folder names that match the product or provider name.
- Update `README.md` when you add a new top-level tool collection.
- Prefer one provider or one documentation topic per pull request.

## Pull Request Checklist

Before opening a PR, verify that:
- file paths are consistent with existing naming
- copied text is formatted cleanly in Markdown
- links resolve correctly
- the PR description explains the source and intent of the update
